更换护照前的准备

在中国驻日本大使馆（六本木的那个）更新护照，
一定网上要预约！
一定网上要预约！
一定网上要预约！

为什么一定要网上预约?

  • 因为中国大使馆3楼办理护照更换的只有一个非预约窗口啊唉！
    ⁽⁽ƪ(- ̠ -)ʃ⁾⁾
  • 因为就算是大使馆官网上说“原则上不再受理非预约申请”，但是去换护照的大多数人（特别是上岁数的人）都是不预约直接就去的啊喂！

  • 网上预约步骤的官方说明
  • 电子护照人像照片的官方要求

但是 ，（注意了！但是！）官方说明还是有几点问题和不明确的地方！

1. 照片尺寸的问题

  • 网上预约时要求电子版照片尺寸为宽33mm x 高48mm 。
  • 但是，在大使馆提交纸质版照片时发现，窗口的漂亮姐姐要的纸质版照片居然不是这个尺寸！(╯‵□′)╯︵┻━┻ 她问我有没有更大的照片。我留意了一下旁边窗口办手续的大哥，大哥的照片居然是正方形的！我当时就紧张了，这个真没有啊！这要是重新拍照然后再排队估计就得明天了好吧！好在漂亮姐姐临场发挥，把我的照片贴到了一张裁好尺寸的白纸上，扔进了扫描仪，然后，扫描成功！厉害了我的姐！（不过这样做我的照片真的不会变得很奇怪么？）
  • 所以提醒各位：在准备护照照片的时候就按照自动照相机预设的护照照片的尺寸准备。作电子版照片时自己在电脑上裁出33mm x 48mm的大小然后上传。纸质版照片直接拿到窗口，别自己裁大小。不然裁不对了，万一再赶上漂亮姐姐心情不好让重拍就有的好看了。

2. 取消网上预约的次数限制

  • 网上预约好之后，如果日程安排有变或者对上传的照片不满意想换等种种原因想重新预约，之前第一次的预约是可以取消的。
  • 但是 （又来了），网上预约只能取消一次！没错，只能取消一次！真的，只能取消一次！别问我是怎么知道的 /(ㄒoㄒ)/~~
  • 如果取消过一次预约，第二次预约又想取消，系统会告诉你已经不能再取消了，下次预约请在3个月后。。。这时你有两个选择：要么等3个月再预约（着急换护照的肯定等不了3个月）；要么等哪天有空，抓起证件一早就直接冲进大使馆去排队。

3. 打印申请表

  • 网上预约好之后，会显示【您的预约申请已成功！】。这时可以选择【打印】、【下载】、【发送至邮箱】。
  • 推荐把申请表的PDF文件下载到随身设备（手机）里，万一去大使馆的当天忘带了，还可以在大使馆附近的便利店打印出来。（大使馆的打印机貌似不支持从手机传文件打印吧，没确认过。）
  • 在去大使馆办手续之前是要把申请表打印出来的，但是注意了，申请表文件一共是5页，只需要打印前两页（第一页：中华人民共和国护照/旅行证申请表(CPT-301)；第二页：申请人声明），后三页中国驻日本大使馆是不要的。
  • 打印完前两页之后，每页的签字栏和日期栏先空着，去大使馆时带着签字笔，在取号窗口交材料时工作人员会让你当面签字的。

4. 其他材料的准备

  • 护照原件
  • 护照的照片页的复印件 x 1
  • 在留卡原件
  • 在留卡的正反面复印件 x 1

在中国驻日本大使馆

中国驻日本大使馆在这里

1. 过安检

进门安检没啥好说的，和机场安检差不多，兜里东西都掏出来放包里，包过扫描机，人过安全门。

2. 直接上3楼

推荐走楼梯，电梯太慢。

3. 找到网上预约专用取号窗口，排队，交材料，取号。

  • 注意，不是上面图片里电子叫号的窗口，是从楼梯上楼直接看到的那个。有很多人排队的就对了。窗口上有类似【网上预约窗口】的标示。
  • 交材料后，工作人员会让你在该签字的地方签字。
  • 取号后记得把材料也一起拿着。

4. 在电子叫号窗口交材料，登录指纹，在电子签字器上签名，领回执。

  • 旧护照这时是要交出去的，也就是说，在新护照下来前，你手里是没有护照的。拿到新护照之前就别到处野了。
  • 新护照可以选择快递到家或者自取，无法办理加急。
    • 选择快递到家的话，要一个月后才能寄到。
    • 选择自取的话，要15个工作日，也就是3周（如果中间没有节假日的话）

总结

  1. 更换护照一定要网上预约。
  2. 电子版照片尺寸和纸质版照片尺寸一定要搞清楚。
  3. 申请表只需打印前两页。
  4. 新护照最快也要3周才能办好。
  5. 在新护照下来前，你手里是没有护照的。
